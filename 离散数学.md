# 离散数学



## 第一章 命题与命题公式

### 命题连接词

- <font color="#F56C6C">否定（$\llcorner$）</font>：取反

  |  P   | $\llcorner$P |
  | :--: | :----------: |
  |  0   |      1       |
  |  1   |      0       |

- <font color="#F56C6C">合取（$\land$）</font>：看作`并`关系，只有P和Q同时为真，P $\land$ Q 为真，否则为假

  |               P                |               Q                |          P $\land$ Q           |
  | :----------------------------: | :----------------------------: | :----------------------------: |
  |               0                |               0                |               0                |
  |               0                |               1                |               0                |
  |               1                |               0                |               0                |
  | <font color="#F56C6C">1</font> | <font color="#F56C6C">1</font> | <font color="#F56C6C">1</font> |

- <font color="#F56C6C">析取（$\lor$）</font>：看作`或`关系，只有P和Q同时为假，P $\lor$ Q 为假，否则为真

  **同或**：两者不互相排斥，可同时成立 **（同真同假）** 要么都吃鱼，要么都不吃。

  **异或**：两者具有相斥性，不能同时成立 比如：要么……要么…… ($\llcorner$q $\land$  p)  $\lor$ (q $\land$ $\llcorner$p)  **（一真一假）** 要么吃鱼，要么吃肉，只能吃一种；

  |               P                |               Q                |           P $\lor$ Q           |
  | :----------------------------: | :----------------------------: | :----------------------------: |
  | <font color="#F56C6C">0</font> | <font color="#F56C6C">0</font> | <font color="#F56C6C">0</font> |
  |               0                |               1                |               1                |
  |               1                |               0                |               1                |
  |               1                |               1                |               1                |
